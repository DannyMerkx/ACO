This folder contains all the functions used by the ACO.

main.m is the main file for the ACO itself, exp.m calls the main file with different
settings for the parameters and pheromone update strategy and returns the appriopriate
results in variables named Results_XXX

The data scripts folder contains files concerning loading and analysis of the data.

I have not included the bayesian network toolbox because it is not my work. It is necessarry to prepare the data for running the ACO, it is freely availlable on
http://www.cs.utah.edu/~tch/notes/matlab/bnt/docs/bnt_pre_sf.html
